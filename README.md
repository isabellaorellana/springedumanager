SpringEduManager - OTEC Primavera 🎓SpringEduManager es una plataforma integral de gestión académica diseñada para la OTEC Primavera. El sistema permite administrar de manera eficiente la oferta educativa, el registro de estudiantes y la gestión de cursos, proporcionando interfaces diferenciadas según el rol del usuario.🧠 Caso de Estudio: Seguridad y Control de Acceso (RBAC)Este proyecto fue seleccionado para el portafolio debido a su complejidad en la implementación de reglas de negocio y seguridad.1. Descripción del DesafíoEl mayor reto técnico fue implementar un sistema de Control de Acceso Basado en Roles (RBAC). Era crítico garantizar que solo los administradores gestionaran la oferta académica, mientras que los estudiantes solo pudieran visualizar su información, evitando brechas de seguridad en la manipulación de datos sensibles.2. Solución PropuestaSe implementó Spring Security 6 utilizando una arquitectura de seguridad personalizada. Se configuró un SecurityFilterChain para definir reglas de acceso granulares por URL y se utilizó BCrypt para la encriptación de contraseñas, asegurando la protección de las credenciales de los usuarios.3. Justificación de HerramientasSpring Boot 3.2: Elegido por su capacidad de autoconfiguración y rapidez para levantar servicios REST profesionales.MariaDB: Seleccionado por su excelente rendimiento en aplicaciones que requieren una integridad referencial estricta.Thymeleaf: Utilizado para permitir una renderización rápida en el servidor sin añadir la complejidad de un framework de frontend separado.4. Aprendizajes Clave y MétricasOptimización de Consultas: Se profundizó en el manejo de JPA y Relaciones (ManyToOne), logrando mantener tiempos de respuesta bajos incluso con grandes volúmenes de datos.Seguridad Robusta: Se alcanzó una cobertura total de seguridad en los endpoints sensibles del sistema.🚀 Características PrincipalesGestión de Cursos y Usuarios: CRUD completo para el catálogo académico y perfiles de usuario.Seguridad de Nivel Empresarial: Encriptación de datos sensibles con BCrypt.Roles de Usuario Definidos:ADMIN: Acceso total a gestión, reportes y configuración.USER: Acceso a portal de autogestión y consulta de información.Arquitectura Escalable: Exposición de APIs RESTful para futuras integraciones.🛠️ Stack TecnológicoLenguaje: Java 25 (OpenJDK Temurin).Framework: Spring Boot 3.2.0.Persistencia: Spring Data JPA.Base de Datos: MariaDB.Seguridad: Spring Security.Interfaz: Thymeleaf, HTML5 y CSS3.📂 Arquitectura del Proyecto (MVC)El sistema está organizado en una estructura de 4 capas para garantizar mantenibilidad:Modelo (Entities): Mapeo de objetos a la base de datos.Repositorio: Comunicación con la base de datos mediante JPA.Servicio: Lógica de negocio y validaciones del sistema.Controlador: Manejo de peticiones HTTP y endpoints.⚙️ Configuración e InstalaciónClonar el repositorio:Bashgit clone https://github.com/isabellaorellana/springedumanager.git
Base de Datos: Configura tus credenciales en src/main/resources/application.properties.Ejecutar:Bashmvn spring-boot:run
